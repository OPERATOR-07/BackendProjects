# Flask Authentication Backend

A simple backend project built with Python Flask that implements
user registration and login with session-based authentication.
This projecte was mainly based on the backend so less energy was put 
in the front end so dont be supprised if the website(templates) are not 
so attractive

## Features
- User registration
- Login & logout
- Session handling
- HTML templates (Jinja2)

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS
